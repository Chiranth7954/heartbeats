What's Going On Here?

This project takes data from the Google Fit API and matches it with the Spotify API to help users optimize a playlist
best suited for their workout and to motivate them by keeping in tune with beats/minute and individual preferences.

GitHub: https://github.com/emilojkovic/heartbeats
